What's the system?

This system is for 2022 DSD Course.

How to use the system?

Device: You can activate or deactivate device, read, set or clear email, and reset the device(i.e. delete all cloud information of the device) on this webpage.
Algorithm: You can get current available models and their introduction, start to train the model, download base model, reset cloud model to base, delete all model on cloud, synchronize cloud model to device, download model from device, and delete model on device on this webpage.
Calibrate: You can calibrate the model of the device on this webpage.
Help: You can get some help of the system on this webpage.
Admin: This is the entry for admin side of the system.

Guidelines?

Just hover the mouse over the elements on the webpage and read the pop-up tooltips.
